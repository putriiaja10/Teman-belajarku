# Teman-belajarku
Aplikasi ini ditujukan untuk anak usia dini agar mereka dapat belajar huruf, angka, dan warna dengan cara menyenangkan. Menggunakan gambar menarik dan suara, anak-anak bisa berinteraksi dengan konten edukatif sambil bermain. Aplikasi ini cocok untuk pengenalan awal belajar membaca dan berhitung.
